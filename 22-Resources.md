# Chapter 22: Resources

This chapter provides a comprehensive list of resources for further learning, research, and practice in the field of Active Directory security. These resources include books, courses, labs, research papers, blogs, communities, and open-source projects.

## Books

Books are an excellent way to gain deep theoretical knowledge and practical insights into Active Directory security.

- **"The Hacker Playbook 3: Practical Guide to Penetration Testing"** by Peter Kim. Covers advanced penetration testing techniques, including Active Directory exploitation.
- **"Red Team Development and Operations: Zero to Hero"** by Zero-Point Security. A comprehensive guide to red team operations, with a strong focus on Active Directory.
- **"Attacking Active Directory: 0 to Da" by** Zero-Point Security. Specifically focuses on Active Directory attacks and exploitation.
- **"Windows Internals, Part 1"** by Pavel Yosifovich, Mark Russinovich, David Solomon, and Alex Ionescu. Essential for understanding the underlying Windows operating system.
- **"Guide to Deploying Active Directory"** by Microsoft. Official Microsoft documentation for deploying and managing Active Directory.

## Courses

Courses provide structured learning paths and hands-on labs to practice Active Directory security techniques.

- **CRTP (Certified Red Team Professional)** by Zero-Point Security. Focuses on Active Directory attacks and privilege escalation.
- **CRTE (Certified Red Team Expert)** by Zero-Point Security. Advanced course covering complex Active Directory attacks and evasion.
- **CARTP (Certified Active Directory Red Team Professional)** by Zero-Point Security. Comprehensive course on Active Directory red team operations.
- **Offensive Security Certified Professional (OSCP)** by Offensive Security. Broad penetration testing certification that includes Active Directory exploitation.
- **eLearnSecurity Junior Penetration Tester (eJPT)** by eLearnSecurity. Entry-level penetration testing certification with Active Directory modules.

## Labs

Labs provide safe, controlled environments to practice Active Directory security techniques without the risk of disrupting production systems.

### Hack The Box
Hack The Box is an online platform that provides vulnerable machines for penetration testing practice. It includes several Active Directory-focused challenges and environments.
- **Website**: [https://www.hackthebox.com](https://www.hackthebox.com)

### TryHackMe
TryHackMe is an online platform that provides guided learning paths and vulnerable machines for penetration testing practice. It includes several Active Directory-focused rooms and paths.
- **Website**: [https://tryhackme.com](https://tryhackme.com)

### GOAD (Game of Active Directory)
GOAD is a set of scripts to build a vulnerable Active Directory environment for practice. It includes multiple domains, trusts, and misconfigurations.
- **GitHub**: [https://github.com/Orange-Cyberdefense/GOAD](https://github.com/Orange-Cyberdefense/GOAD)

### AD Security Lab
A collection of scripts and guides to build an Active Directory lab for security testing.
- **GitHub**: [https://github.com/cosine-security/AD-Security-Lab](https://github.com/cosine-security/AD-Security-Lab)

## Research Papers

Research papers provide in-depth analysis and new discoveries in the field of Active Directory security.

- **"Certified Pre-Owned: Abusing Active Directory Certificate Services"** by SpecterOps. The definitive paper on AD CS attacks (ESC1-ESC10).
- **"Kerberos Golden Ticket: A Comprehensive Guide"** by various authors. Detailed analysis of Golden Ticket attacks and mitigations.
- **"Delegation Attacks in Active Directory"** by various authors. Analysis of unconstrained, constrained, and resource-based delegation attacks.
- **"Abusing Active Directory ACLs"** by various authors. Analysis of ACL misconfigurations and privilege escalation.

## Open-Source Projects

Open-source projects provide tools, scripts, and frameworks for Active Directory security assessments.

- **BloodHound / SharpHound**: [https://github.com/BloodHoundAD/BloodHound](https://github.com/BloodHoundAD/BloodHound) - Graph-based AD analysis tool.
- **Mimikatz**: [https://github.com/gentilkiwi/mimikatz](https://github.com/gentilkiwi/mimikatz) - Credential extraction and manipulation tool.
- **Rubeus**: [https://github.com/GhostPack/Rubeus](https://github.com/GhostPack/Rubeus) - Kerberos interaction and abuse tool.
- **Impacket**: [https://github.com/SecureAuthCorp/impacket](https://github.com/SecureAuthCorp/impacket) - Python classes for working with network protocols.
- **PowerSploit / PowerView**: [https://github.com/PowerShellMafia/PowerSploit](https://github.com/PowerShellMafia/PowerSploit) - PowerShell post-exploitation framework.
- **Certipy**: [https://github.com/ly4k/Certipy](https://github.com/ly4k/Certipy) - AD CS exploitation tool.
- **Responder**: [https://github.com/lgandx/Responder](https://github.com/lgandx/Responder) - LLMNR/NBT-NS poisoning and NTLM relay tool.
- **CrackMapExec / NetExec**: [https://github.com/byt3bl33d3r/CrackMapExec](https://github.com/byt3bl33d3r/CrackMapExec) - Network service exploitation tool.
